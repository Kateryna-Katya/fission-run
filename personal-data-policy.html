Политика обработки персональных данных

1. Общие положения

Настоящая политика определяет порядок и условия обработки персональных данных, предпринимаемые проектом Fission Run (далее — «Оператор»), и устанавливает меры по обеспечению безопасности этих данных. Настоящая Политика применяется ко всей информации, которую Оператор может получить о посетителях веб-сайта Fission Run.

2. Основные понятия

  • Пользователь — любой посетитель веб-сайта.
  • Персональные данные — любая информация, относящаяся к определенному Пользователю.
  • Обработка данных — любые действия с персональными данными (сбор, хранение, передача и т.д.).

3. Данные, которые мы обрабатываем

Оператор может обрабатывать следующие категории данных:

  • Данные, предоставляемые Пользователем: Имя, адрес электронной почты, номер телефона.
  • Данные, собираемые автоматически: IP-адрес, тип браузера, cookies, данные статистики.

4. Цели обработки

  • Идентификация Пользователя.
  • Установление обратной связи (ответы на заявки).
  • Улучшение качества работы сайта и аналитика.

5. Безопасность и хранение

Оператор обеспечивает сохранность данных и принимает все меры для их защиты. Персональные данные никогда не будут переданы третьим лицам, за исключением случаев, предусмотренных законодательством.

6. Контакты

Пользователь может получить любые разъяснения по вопросам обработки своих данных, обратившись к Оператору по электронной почте: hello@fission-run.blog.
